# Tetris-written-in-Rust
- The classic Tetris - written in Rust, with pure vibe coding in Claude 3.7

## Dependencies
### Install Rust
```curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh```

## Clone the repo
```git clone https://github.com/meerkatone/rust_tetris.git```

## Build
```cd rust_tetris```\
```cargo run```

## Game Controls

Up arrow (↑): &uarr;\
Down arrow (↓): &darr;\
Left arrow (←): &larr;\
Right arrow (→): &rarr;
